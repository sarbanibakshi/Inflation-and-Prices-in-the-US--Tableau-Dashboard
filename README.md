# Inflation-and-Prices-in-the-US--Tableau-Dashboard
Interactive Tableau dashboard designed to analyze and visualize trends in inflation and consumer prices from 2005 to 2025 for the US economy.

The dashboard transforms raw economic data into clear, actionable insights using dynamic visualizations and intuitive filters.

The goal of this project is to:

* Explore long-term inflation patterns for the 20 year duration.

* Compare movements in CPI vs Core CPI.

* Show comparison between the inflationary expectations and the actual inflation.

* Present the changes in the different indices to understand the movements of the US economy.

The dashboard intends to show:

* Historical trends and fluctuations in inflation.

* Comparative price changes across years for various segments of the market.

* Key patterns, spikes, and periods of stability for the US economy.

* Interactive views that allow to the users to understand how the changes in price indices are correlated across the segments. 

Data source:(https://fred.stlouisfed.org/)
